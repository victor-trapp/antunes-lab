# Mini Coxinhas

Landing page em React + TypeScript + Tailwind CSS.

## Rodando localmente

```bash
npm install
npm run dev
```

## Build de produção

```bash
npm run build
npm run preview
```

## Estrutura principal

- `src/App.tsx`: layout e componentes da página
- `src/index.css`: estilos globais e utilitários extras
- `public/images/`: imagens do site
